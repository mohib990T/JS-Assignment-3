1. What are the differences between var, let and const?

Ans :-

var :-

  1. Variables declared with var are in the function scope.
  2. We can declare var variables after defining them.
  3. We can define new values to the var variables
  4. We allowed to declare the same variable again.

let :-

  1. Variables declared with let are in the block scope.
  2. let variables do not support hoisting.
  3. We can define new values to the let variables
  4. let variables do not allow you to redeclare them again.

const :-

  1. Variables declared with const are in the block scope.
  2. const variables do not support hoisting.
  3. const variables value cannot be reassigned.
  4. const variables do not allow you to redeclare them again.

9. Explain the Pre-Increment and Post Increment in js.

Ans :-

Pre-increment (++i) - Before assigning the value to the variable, the value is incremented by one.
Ex :- let x = 3;
let y = ++x;
console.log(x) //it will give output 4
console.log(y) //it will give output 4.
Post-increment (i++) - After assigning the value to the variable, the value is incremented.
Ex :- let x =3;
let y = x++;
console.log(x) //it will give output 4
console.log(y) //it will give output 3

10. Difference between “ == “ and “ === “ operators.

Ans :-

The "==" operator is the comparison operator. It will only return true if both values are equivalent after checking their types to the same type.
The "===" operator is a more strict comparison operator often called the identity operator. It will only return true if both the type and value of the operands are the same.
